# DESIGN SYSTEM Epic - VISION: Glassmorphism Design Tokens

## 1. Aesthetic Vision

The K.A.O.S Design System provides a unified, reusable kit of components styled with modern glassmorphism. It replaces all custom CSS overrides with theme tokens and utility classes, establishing a futuristic, unified desktop experience.

### Core Visual Guidelines:
- **Depth:** Elements hover over a dark slate void (`hsl(240, 10%, 4%)`) using backdrop blurs and subtle drop shadows.
- **Micro-signals:** Colors indicate status:
  - Green (`hsl(142, 70%, 45%)`) for active, healthy, and completed items.
  - Purple (`hsl(270, 70%, 60%)`) for cognitive processing and thinking.
  - Red (`hsl(0, 72%, 50%)`) for emergency faults and blocks.
  - Amber (`hsl(38, 92%, 50%)`) for warnings and degraded state.
- **Tactility:** Cards transition borders on hover to feel clickable and alive.
