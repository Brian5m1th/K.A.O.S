# UX Epic - REFERENCES: Front-End Inspiration

This document catalogs references and tutorials for front-end architecture.

---

## 1. Architecture Methodology

- **Feature-Sliced Design (FSD):** [FSD Official Guide](https://feature-sliced.design/). The structural standard for modern React applications.
- **Zustand State Store:** [Zustand Documentation](https://github.com/pmndrs/zustand). Minimalist state manager for React hooks.

---

## 2. Visual Frameworks

- **Framer Motion API:** [Framer Motion Docs](https://www.framer.com/motion/). Animation library for components.
- **Glassmorphism CSS Generators:** [Glassmorphism UI Patterns](https://cssglass.com/).
- **Lucide Icons:** Premium, modern icon set used across the interface.
