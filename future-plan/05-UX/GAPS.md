# UX Epic - GAPS: Front-End Usability Backlog

The following design gaps must be addressed to transition the user experience:

| UX Gap | Severity | Technical Debt | User Impact |
| :--- | :--- | :--- | :--- |
| **Separated Feature Pages** | **High** | High | User has to jump across 21 pages. Needs consolidation into a unified Mission view. |
| **No State Animation** | Low | Low | The interface feels static and unresponsive. Needs Framer Motion transitions. |
| **No Desktop Toasts** | High | Medium | Important alerts (e.g. build errors, email triggers) are invisible unless Tauri is open. |
| **Complex Settings** | High | Low | User must input complex API keys and paths manually. Needs a simplified profile wizard. |
| **Static Command Palette** | Medium | Medium | CTRL+K palette only searches pages; doesn't execute cognitive shortcuts. |
